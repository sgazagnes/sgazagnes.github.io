This page provides an overview of the only project related to Biology that I worked on. From March to August 2016, I worked in the Morpheme Team​, which is part of the Inria laboratory I3S, under the supervision of Emmanuel Soubies and Laure Blanc-Féraud. My project was related to the implementation of a new algorithm to perform high-precision molecule localization using photo-activated localization microscopy techniques (PALM). This page describes this project. The different sections were first written by myself, then put into ChatGPT with some weird requests.
